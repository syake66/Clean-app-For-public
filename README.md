# Clean Management App (Public Portfolio)

Next.js と Supabase を活用した、実用的な清掃スタッフ・シフト管理システムです。

## 🌟 プロジェクトの概要
このシステムは、民泊運営における「清掃管理の漏れ」と「煩雑なシフト調整」を解決するために開発されました。Googleカレンダーの予約情報を自動で取り込み、スタッフがモバイルから即座に報告できる仕組みを提供します。

### 🚀 主な機能
- **Google Calendar 自動同期**: 予約予定を自動的にタスク化。
- **インテリジェント競合回避**: 現場での手動変更を保護する同期アルゴリズム。
- **リアルタイムダッシュボード**: スタッフの報告を管理者が即座に確認。
- **モバイル最適化**: 現場スタッフが使いやすい直感的なUI。

## 🛠 技術スタック
- **Frontend**: Next.js (App Router), TypeScript, Tailwind CSS
- **Backend**: Supabase (PostgreSQL, Auth, Storage, Edge Functions)
- **Deployment**: Vercel

## 🤖 AIとの協働開発 (AI-Human Collaboration)
このプロジェクトは、最新のAIエージェントを活用した「高速かつ高品質な開発プロセス」の実証実験でもあります。
- **複雑なロジックの共同設計**: カレンダー同期におけるエッジケースの解消。
- **セキュアな設計**: Supabase RLSの設定や認証フローのAIレビュー。

## ⚙️ セットアップ
1. リポジトリをクローン
2. `.env.example` をコピーして `.env.local` を作成し、自身のキーを設定
3. `npm install`
4. `npm run dev`
